# Echo-Slack-Bot

This bot can be added to your slack conversations and be extremly helpful!

# Features: 
/dsb-ping-echo: This tests the latency to the bot 
/dsb-catfact-echo: Echo loves giving out fun cat facts! 
/dsb-joke-echo: Echo will try their very best to tell a joke 
/dsb-help-echo
